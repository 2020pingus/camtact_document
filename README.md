>
> 감독자 클라이언트: https://github.com/2020pingus/camtact_manager



</br>



## ✍️ 주요 기능

- ##### 360도 회전캠 감독

  - 감독자가 응시자의 회전캠을 360도 원격 조정하면서 응시자의 사각지대를 최소화

- ##### 화면 스트리밍

  - 회전캠을 통해 감독자가 응시자의 상태를 감시

- ##### 화면 녹화

  - 회전캠을 통해 응시자의 상태를 녹화한 뒤, 감독자가 후에도 확인 가능

- ##### 온라인 시험 응시

  - 온라인으로 시험을 응시할 수 있는 환경을 제공
